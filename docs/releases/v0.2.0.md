# VarjoToolkit v0.2.0

Rendering, package, and video post process sample release.

## Highlights

- CMake install/package support with `find_package(VarjoToolkit CONFIG REQUIRED)`.
- Package consumer CTest coverage.
- HMD-required tests are separated behind `VARJOTOOLKIT_BUILD_HMD_TESTS`.
- OpenCppCoverage helper script.
- D3D11 and D3D12 minimal render samples.
- Varjo Native SDK experimental Video Post Process Shader wrapper.
- D3D11 and D3D12 video post process samples.
- Occlusion mesh CSV/OBJ dump sample.
- Event and marker tracking services with CSV output.

## Validation commands

```bat
set "VARJO_SDK_ROOT=C:\work\library\varjo-sdk-experimental"
set "PATH=%VARJO_SDK_ROOT%\bin;%PATH%"

git pull

rmdir /s /q out\build\release-check 2>nul

cmake -S . -B out\build\release-check ^
  -G "Visual Studio 17 2022" ^
  -A x64 ^
  -DVARJO_SDK_ROOT="%VARJO_SDK_ROOT%" ^
  -DVARJOTOOLKIT_BUILD_SAMPLES=ON ^
  -DVARJOTOOLKIT_BUILD_TESTS=ON ^
  -DVARJOTOOLKIT_BUILD_HMD_TESTS=OFF ^
  -DVARJOTOOLKIT_ENABLE_EXPERIMENTAL_MR_POSTPROCESS=ON

cmake --build out\build\release-check --config Release
ctest --test-dir out\build\release-check -C Release --output-on-failure
```

## Optional HMD validation

```bat
rmdir /s /q out\build\hmd-check 2>nul

cmake -S . -B out\build\hmd-check ^
  -G "Visual Studio 17 2022" ^
  -A x64 ^
  -DVARJO_SDK_ROOT="%VARJO_SDK_ROOT%" ^
  -DVARJOTOOLKIT_BUILD_SAMPLES=ON ^
  -DVARJOTOOLKIT_BUILD_TESTS=ON ^
  -DVARJOTOOLKIT_BUILD_HMD_TESTS=ON ^
  -DVARJOTOOLKIT_ENABLE_EXPERIMENTAL_MR_POSTPROCESS=ON

cmake --build out\build\hmd-check --config Release
ctest --test-dir out\build\hmd-check -C Release -L hmd --output-on-failure
```

## Known limitations

- D3D rendering samples are minimal clear/triangle samples.
- Video post process samples demonstrate constant-buffer-only effects. Additional shader texture input samples are not included yet.
- Occlusion mesh GPU buffer / stencil integration is not included yet.
